Notes for review
  1. Cards are sorted by last updated; the tab badge shows the count.
  2. Thumbnail = one mini map per board (same SVG technique as the Maps tab previews) + team-colored dots for occupied tiles. Character portraits are a possible v2 refinement.
  3. Select switches the mode picker to the team's mode, loads the snapshot into the boards, and overwrites that mode's active team (toast: "Team loaded"). No modal — consistent with the app's toast-first style.
  4. Delete is a two-step inline confirm (arms for 3 s) instead of a modal dialog.
  5. Duplicate copies the snapshot as "<name> (copy)".
  6. Rename is inline (click the name; Enter/blur commits, Esc cancels) — this is the "Edit" affordance.
  7. Cap indicator appears in the header once the library reaches 80% of the 50-team soft cap.
  8. On mobile this panel lives in the same pull-up sheet as the other roster tabs; selecting a team collapses the sheet so the loaded boards are visible (same pattern as placing a character).
