Notes for review (Rev 2)
  1. The outer tab is renamed 5 v 5 → Teams (existing i18n key app.teams).
  2. Mode picker now has four segments: 1v1 · 3v3 · 5v5 · 5v5 Supreme League, fed by the TEAM_MODES registry — a future mode is one registry entry, no new UI.
  3. The active team label shows which saved team you're editing ("Unsaved team" when none); the amber dot marks unsaved changes. Try it: click Save (dot clears), or switch modes (each mode remembers its own team).
  4. Save updates the selected team in place; Save as New opens the name popover (prefilled auto-name, max 60 chars, Enter/Esc) and creates a copy. Try both.
  5. Switching modes is instant and lossless: each mode autosaves to its own localStorage slot, so no confirmation dialog is needed.
  6. Wrap renders only for 5-board modes; 1 or 3 boards always fit one row.
  7. Default maps: 1v1, 3v3, and 5v5 = Arena I on every board; 5v5 Supreme League keeps today's list (Arena I–III, Preset SR11, SR1).
  8. Export/Import operate on the saved-teams library as a JSON backup file (see saved-teams-tab.html).
  9. Everything below the new row is untouched (display toggles, invert, link/copy/download/clear, boards, per-board actions).
